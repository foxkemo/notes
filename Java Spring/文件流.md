# 1. 文件输入流 FileInputStream

从文件读取数据。

```java
FileInputStream fis =
        new FileInputStream("a.txt");
```

---

## 读取文件

```java
import java.io.FileInputStream;

public class Main {
    public static void main(String[] args) throws Exception {

        FileInputStream fis =
                new FileInputStream("a.txt");

        int data;

        while ((data = fis.read()) != -1) {
            System.out.print((char) data);
        }

        fis.close();
    }
}
```

---

# read()

```java
fis.read()
```

返回：

- 一个字节
    
- EOF 返回 -1

# 五、文件复制

```java
import java.io.*;

public class Main {

    public static void main(String[] args) throws Exception {

        FileInputStream fis =
                new FileInputStream("a.jpg");

        FileOutputStream fos =
                new FileOutputStream("b.jpg");

        byte[] buffer = new byte[8192];

        int len;

        while ((len = fis.read(buffer)) != -1) {
            fos.write(buffer, 0, len);
        }

        fis.close();
        fos.close();
    }
}
```

# 八、Spring Web 文件下载（重点）

本质：

```text
服务器 -> OutputStream -> 浏览器
```

---

# 方式1：直接写 HttpServletResponse

最经典。

```java
@GetMapping("/download")
public void download(
        HttpServletResponse response)
        throws Exception {

    File file = new File("a.pdf");

    response.setContentType(
        "application/pdf");

    response.setHeader(
        "Content-Disposition",
        "attachment;filename=a.pdf");

    FileInputStream fis =
            new FileInputStream(file);

    ServletOutputStream os =
            response.getOutputStream();

    byte[] buffer = new byte[8192];

    int len;

    while ((len = fis.read(buffer)) != -1) {
        os.write(buffer, 0, len);
    }

    fis.close();
}
```

---

# Content-Disposition

```http
attachment
```

表示：

```text
触发浏览器下载
```

# 九、Spring Boot 推荐下载方式

---

# ResponseEntity

更现代。

```java
@GetMapping("/download")
public ResponseEntity<Resource> download()
        throws Exception {

    File file = new File("a.pdf");

    InputStreamResource resource =
            new InputStreamResource(
                    new FileInputStream(file));

    return ResponseEntity.ok()
            .header(
                HttpHeaders.CONTENT_DISPOSITION,
                "attachment;filename=a.pdf")
            .contentLength(file.length())
            .body(resource);
}
```

# 十八、一个完整现代下载示例

```java
@GetMapping("/download")
public ResponseEntity<Resource> download()
        throws Exception {

    Path path = Path.of("test.pdf");

    Resource resource =
            new InputStreamResource(
                    Files.newInputStream(path));

    return ResponseEntity.ok()
            .header(
                    HttpHeaders.CONTENT_DISPOSITION,
                    "attachment; filename=test.pdf")
            .contentType(
                    MediaType.APPLICATION_PDF)
            .body(resource);
}
```
