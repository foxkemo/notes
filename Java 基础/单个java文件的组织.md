

| **情况**                 | **限制规则**                                                    |
| ---------------------- | ----------------------------------------------------------- |
| **存在 public 类**        | 必须**有且仅有**一个；文件名必须与该类名完全一致。                                 |
| **不存在 public 类**       | 可以有多个 `default` 类；文件名可以不匹配任何类名。                             |
| **内部类（static or not）** | 不受上述限制。一个 `public` 顶级类里可以嵌套无数个 `private` 或 `public` 的静态内部类。 |
