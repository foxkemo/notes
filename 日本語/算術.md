在日语中，表达加、减、乘、除的基本运算主要有两套系统：一种是较为口语/实用的「〜は（〜になる）」**形式，另一种是数学专用的**「算式读法」。

以下是具体的表达方式：

### 1. 加法 (足し算 - たしざん)

- **常用动词：** 足す (たす)
    
- **句型：** A **たす** B **は** C
    
- **范例：** $1 + 2 = 3$
    
    - いち **たす** に **は** さん。 (1加2等于3)
        

### 2. 减法 (引き算 - ひきざん)

- **常用动词：** 引く (ひく)
    
- **句型：** A **ひく** B **は** C
    
- **范例：** $5 - 3 = 2$
    
    - ご **ひく** さん **は** に。 (5减3等于2)
        

### 3. 乘法 (掛け算 - かけざん)

- **常用动词：** 掛ける (かける)
    
- **句型：** A **かける** B **は** C
    
- **范例：** $4 \times 3 = 12$
    
    - よん **かける** さん **は** じゅうに。 (4乘以3等于12)
        
- **注意：** 日本也有类似“九九乘法口诀”的读法（九九 - くく），读法会为了押韵而发生音变，但在一般描述算式时，使用「かける」即可。
    

### 4. 除法 (割り算 - わりざん)

- **常用动词：** 割る (わる)
    
- **句型：** A **わる** B **は** C
    
- **范例：** $10 \div 2 = 5$
    
    - じゅう **わる** に **は** ご。 (10除以2等于5)
        

---

### 5. 其他相关数学符号

|**中文含义**|**符号**|**日语读法**|**备注**|
|---|---|---|---|
|**等于**|$=$|は / イコール (equal)|口语常用「は」，正式场合用「イコール」|
|**余数**|...|あまり|例：$7 \div 3 = 2$ 余 $1$ (2あまり1)|
|**小数点**|$.$|点 (てん)|例：$3.14$ (さん・てん・いち・よん)|
|**分数**|$\frac{B}{A}$|**A** 分の (ぶんの) **B**|**先读分母**。例：$\frac{2}{3}$ (さんぶんのに)|
|**正负号**|$\pm$|プラス / マイナス|例：$-5$ (マイナスご)|

### 场景示例

如果你在编写技术文档、查阅论文或者进行学术报告（正如你正在进行的 Java AI 动态研究或计算机课程报告），通常会用到更正式的表达：

- **算式表达：** 「AにBを加算（かさん）する」(给A加算B)
    
- **减法表达：** 「AからBを減算（げんさん）する」(从A减去B)
    
- **程序逻辑：** 「$i$ を $1$ ずつインクリメント（increment）する」(让 $i$ 逐次递增 1)
    

**小贴士：** 在日语输入法中，输入「たす」、「ひく」、「かける」、「わる」通常可以直接变换出对应的数学符号「$+$」、「$-$」、「$\times$」、「$\div$」。
